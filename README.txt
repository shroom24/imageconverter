This small Flask application converts an image into a text file for import with the DAK module of the Kniterate editor.

It has been tested and run with AWS Beanstalk

Use white background for empty stitches and black and yellow as colors for different stitches. There is a test file in the image directory.

Optons are: 

• repeating the rows in the pattern so it can be imported as a shaped reverserd jacquard
• adding empty lines between shaped rows for easier editing of transfers for the shaping
